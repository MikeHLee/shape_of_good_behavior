# ICML 2025 Submission: Sheaf-Theoretic Reward Spaces

This directory contains the manuscript and resources for the paper **"Sheaf-Theoretic Reward Spaces for Safe Reinforcement Learning"**.

## Directory Structure

*   `DRAFT_PAPER.md`: Full consolidated manuscript draft.
*   `sections/`: Individual Markdown files for each section of the paper (Introduction, Methods, Experiments, etc.).
*   `PAPER_OUTLINE.md`: Initial detailed outline and planning document.
*   `CONFERENCE_CALENDAR.md`: Tracking for target venues and deadlines.

## Reproducing Experiments

The experimental results in Section 6 are generated by the following Jupyter notebooks located in the parent directory:

1.  **Condorcet Cycle (H¹ Detection)**:
    *   File: `../Condorcet_Cycle_Experiment.ipynb`
    *   Verifies that SGPO's Hodge Critic correctly learns the harmonic component $\omega$ corresponding to the cyclic preference structure.

2.  **Geometric Safety (Black Hole Avoidance)**:
    *   File: `../AI_Safety_Benchmark.ipynb`
    *   Demonstrates SGPO's ability to avoid catastrophic states via Riemannian metric learning, comparing against PPO and CPO.

3.  **LLM Style Cycling**:
    *   File: `../LLM_Style_Tuning_Experiment.ipynb`
    *   Simulates navigating a continuous style manifold with cyclic user preferences.

### Running the Code

To run the experiments and generate the data for the paper:

1.  Navigate to the root directory `high_dimensional_reward_spaces/`.
2.  Install dependencies:
    ```bash
    pip install torch numpy matplotlib jupyter
    ```
3.  Execute the notebooks to generate JSON metrics files (`*_metrics.json`).
4.  Run the aggregation script to summarize results:
    ```bash
    python aggregate_metrics.py
    ```

## Building the PDF

The `DRAFT_PAPER.md` is formatted in standard Markdown. To generate a PDF for submission:

1.  Use `pandoc` with a standard academic template:
    ```bash
    pandoc DRAFT_PAPER.md -o submission.pdf --bibliography=references.bib
    ```
    *(Note: `references.bib` needs to be populated with citations)*
